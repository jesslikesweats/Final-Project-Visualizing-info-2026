Project Question

The question I looked at was:
Do heavier cars use more gas?

Dataset

I used the Auto MPG dataset. It has information about cars like MPG, weight, horsepower, cylinders, and other things. Each row is a different car.

What the Data Looks Like

Here are the first few columns in the dataset: mpg, cylinders, displacement, horsepower, weight, acceleration, model year, origin, and car name.

Design Process

At first I was not sure what chart to use. I tried thinking about bar charts, but they did not really show relationships between variables. Then I chose a scatterplot because it shows how two numbers compare, also i think a scatterplot is more interactive with the pointer, which would benefit my design.

Design Choices

I put car weight on the x-axis and MPG on the y-axis. I did this because I wanted to see if heavier cars use more gas. I also added tooltips so when you hover over a dot, you can see the exact values. I used a scatterplot because my goal was to compare two numerical variables: car weight and miles per gallon (MPG). A scatterplot is the best chart type for showing relationships between two continuous variables. I placed weight on the x-axis and MPG on the y-axis because weight is the independent variable I wanted to compare against fuel efficiency. MPG depends on how heavy a car is, so it made sense to put it on the vertical axis. I used circles for each data point because each circle represents one car, and this makes it easy to see patterns in the full dataset. I also used color (blue) for all points so the focus stays on the relationship between the variables instead of extra categories. Since I was not comparing groups, using one color keeps the visualization simple and clear. Tooltips were added so users can hover over a point and see exact values. This helps with detailed exploration without cluttering the chart.

What I Found

To explore the data, I looked at how the points are distributed on the graph. I noticed that cars with higher weight values tend to have lower MPG values. For example, heavier cars (around 3500–4500 lbs) usually fall below 20 MPG, while lighter cars often reach 30–40 MPG. This pattern shows a negative relationship between weight and fuel efficiency, meaning that as car weight increases, MPG tends to decrease.
